# train.py
- **1단계 : 시뮬레이터 구동**
    1. AppLauncher(명령어 옵션 : 헤드리스 모드)로 Isaac Sim 실행
    2. 이 엔진이 켜져야 비로소 gym, torch 같은 라이브러리들이 정상적으로 학습 루프를 돌 수 있는 환경 준비
- **2단계 : 설정의 확정**
    1. hydra_task_config 로 미리 작성된 환경 설정 파일 + 에이전트 설정 파일을 로드
    2. 단순 로딩이 아니라, 터미널에서 넘긴 num_envs, seed 같은 인자를 이용해 기본 설정 덮어쓰기
- **3단계 : 환경 생성과 포장**
    - gym.make(...) 호출 → 로봇과 지형이 포함된 시뮬레이션 환경 생성
    - RslRlVecEnvWrapper 로 Raw data(관측/보상/Done 신호)를 강화학습 알고리즘의 입력 데이터로 변환
- **4단계 : 학습 시작**
    - OnPolicyRunner 는 실제 PPO 알고리즘을 실행하는 코치(학습기) 역할
    - 학습 시작 전, 모든 설정값을 로그 폴더에 저장(재현성, 디버깅 기록 목적)
    - runner.learn() 호출 → 수천·수만 번의 에피소드/스텝을 반복하며 정책을 학습

# spot_flat_env_cfg.py
**로봇 입출력 정의**: 관절 위치(Joint Position)로 움직이고, 몸체 속도·자세 등 센서 정보를 관측값(Observation)으로 받도록 설정합니다.

**목표 설정(Command)**: "앞으로 xm/s로 가라", "yaw를 얼마나 돌려라"와 같은 속도 명령의 범위와 형태를 정의합니다.

**보상 체계(Reward)**: 전진 속도, 발을 끌지 않기(Foot clearance), 부드럽게 걷기(Smoothness), 특정 보행 패턴 유지(Gait 안정성) 등을 모두 보상으로 유도합니다.

**현실감 부여(Domain Randomization)**: 마찰력, 질량, 외부 충격 등을 랜덤하게 바꿔서 환경 변화에 강인한(Robust) 정책을 만들도록 유도합니다.
